Este aplicativo possibilita ao Android memorizar o volume de diferentes dispositivos Bluetooth.
O volume da música e da chamada podem ser configurados para cada dispositivo.
A sincronização e a configuração para cada dispositivo Bluetooth podem ser ajustadas individualmente.

Permissões justificadas:
• 'Internet' para relatórios de erros.
• 'Bluetooth' para emparelhar com dispositivos bluetooth.
• 'Modificar configurações de áudio' para alterar o volume.
• 'Inicialização concluída' para restaurar o volume após a reinicialização.
• 'WAKE_LOCK' para solucionar uma falha nos dispositivos Samsung, onde esta permissão é necessária para evitar um erro.
