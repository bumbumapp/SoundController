Приложение позволяет системе Андроид запомнить громкость различных устройств Bluetooth.
Для каждого устройства можно настроить как громкость вызова, так и громкость музыки.
Временные показатели и конфигурация для каждого устройства Bluetooth могут быть настроены индивидуально.

Объяснение разрешений:
• 'Internet (Интернет)' для сообщений об ошибках.
• 'Bluetooth' для работы с устройствами Bluetooth.
• 'Modify audio settings (Изменить настройки аудио)' для изменения уровня громкости.
• 'Boot completed (Загрузка выполнена)' для восстановления громкости после перезагрузки.
• 'WAKE_LOCK (ВЭЙКЛОК)' для устранения сбоев на устройствах Samsung, где это разрешение необходимо, чтобы избежать ошибки.
